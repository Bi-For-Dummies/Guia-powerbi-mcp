Conceptos básicos

¿Qué es el MCP y para qué sirve?

Antes de instalar nada, entendamos qué vamos a hacer y por qué es tan útil.

✏️ Explicación para todos

Imagina que Claude es un asistente brillante que sabe todo sobre datos, pero solo puede hablar. El MCP es como darle manos para que pueda abrir tu Power BI, leer tu modelo y trabajar directamente en él — todo desde una conversación en español.

🤖

Claude Desktop

La app de escritorio de Claude AI. Aquí escribes tus instrucciones en español normal, como si chatearas con un experto en datos disponible 24/7.

🔌

MCP (Model Context Protocol)

El protocolo que conecta a Claude con aplicaciones externas. Piénsalo como el cable USB que permite que dos dispositivos se comuniquen entre sí.

📊

Power BI Desktop

La herramienta de visualización de Microsoft. Con esta integración, Claude puede analizar tu modelo y crear medidas DAX sin que toques una línea de código.

✨

Tutorial paso a paso

Instalación completa en 5 pasos

Tiempo estimado: 10 minutos. Sin necesidad de saber programar.

⚠️ Requisitos previos: Debes tener instalados Claude Desktop, Power BI Desktop y Visual Studio Code. Los tres son gratuitos.

1
📦 Instalar la extensión MCP en VS Code

Abre Visual Studio Code, ve al panel de extensiones (ícono de cuadros en la barra lateral izquierda) y busca exactamente esto:

Búsqueda en VS Code
Power BI Modeling MCP (Analysis Services)
💡 Instala la extensión y luego busca la carpeta donde quedó instalada. La necesitarás en el paso 3.
2
🔍 Encontrar la ruta exacta del ejecutable

Abre PowerShell (búscalo en el menú inicio de Windows) y ejecuta este comando. Te devolverá tu ruta automáticamente:

PowerShell
# Pega esto en PowerShell y presiona Enter Get-ChildItem "$env:USERPROFILE\.vscode\extensions" | Where-Object {$_.Name -like "*powerbi*"} | Select-Object FullName

El resultado se verá parecido a esto. Copia esa ruta completa:

Ejemplo de resultado
C:\Users\TuUsuario\.vscode\extensions\ analysis-services.powerbi-modeling-mcp-X.X.X-win32-x64
✅ A esa ruta agrégale al final: \server\powerbi-modeling-mcp.exe
3
⚙️ Configurar el archivo JSON de Claude Desktop

En Claude Desktop ve a Settings → Developer → Edit Config. Se abre el archivo claude_desktop_config.json. Reemplaza TODO su contenido con esto, cambiando TuUsuario y la versión X.X.X:

claude_desktop_config.json
{ "mcpServers": { "powerbi-modeling-mcp": { "command": "C:\\Users\\TuUsuario\\.vscode\\extensions\\analysis-services.powerbi-modeling-mcp-X.X.X-win32-x64\\server\\powerbi-modeling-mcp.exe", "args": ["--start"], "type": "stdio" } } }
⚠️ En Windows las rutas usan barra doble \\ en el JSON. Si pones una sola \ verás el error "Bad escaped character". ¡Es el error más común!
💡 Modo solo lectura (recomendado para empezar): Cambia "--start" por "--start", "--readonly" para que Claude solo pueda ver sin modificar nada.
4
🔄 Reiniciar Claude Desktop correctamente

Este paso es crítico. Cerrar la ventana no es suficiente — debes cerrar el proceso completamente desde la bandeja del sistema.

Cómo cerrar Claude Desktop completamente
1️⃣Busca el ícono de Claude en la bandeja del sistema (esquina inferior derecha)
2️⃣Haz clic derecho sobre el ícono de Claude
3️⃣Selecciona "Quit" o "Salir"
4️⃣Abre Claude Desktop nuevamente desde el menú inicio
5
✅ Verificar la conexión — Botón + → Conectores

Abre un archivo .pbix en Power BI Desktop. Luego en Claude Desktop haz clic en el botón "+" del chat y selecciona "Conectores". Verás powerbi-modeling-mcp disponible para activar.

Claude Desktop — Botón + → Conectores
➕Haz clic en el botón "+" en la barra inferior del chat
🔌Selecciona "Conectores" en el menú que aparece
✅Verás powerbi-modeling-mcp con el toggle azul activado
💬¡Escribe en el chat! Claude ahora puede ver y modificar tu modelo
🎉 ¡Si ves powerbi-modeling-mcp activado en Conectores, la integración está lista y funcionando!

Comparación

Desktop vs. Navegador Web

¿Puedes usar esta integración desde el navegador? Aquí están las diferencias clave.

💡 La razón técnica

El MCP de Power BI Desktop corre un servidor local en tu computador. El navegador no puede acceder a procesos locales por seguridad — es como intentar llamar a un teléfono que solo existe dentro de tu casa desde afuera.

MCP con Power BI Service (nube)✅ Sí🔄 En desarrollo

Analizar estructura del modelo✅ Sí❌ No
Gmail / Google Calendar✅ Sí✅ Sí

Pide documentación

Claude puede documentar automáticamente todas tus medidas. Perfecto para cuando se integra alguien nuevo al equipo.

Preguntas frecuentes

¿Tienes dudas?

Las preguntas más comunes respondidas.

¿Necesito saber programar para usar esto?▼
No necesitas saber programar en absoluto. Una vez instalado, solo escribes en español normal lo que quieres y Claude lo hace por ti. Es como tener un experto en Power BI disponible 24/7.
¿Es seguro? ¿Mis datos salen de mi computador?▼
El servidor MCP corre completamente en tu máquina local. Los datos de tu modelo de Power BI no salen de tu computador. Lo que sí va a los servidores de Anthropic es el texto de tu conversación con Claude, igual que cualquier chatbot.
¿Por qué me sale el error "Bad escaped character"?▼
Es el error más común. En el archivo JSON las rutas de Windows con "\" deben escribirse con doble barra "\\". Por ejemplo: C:\Users debe ser C:\\Users. Revisa que cada barra en tu ruta esté duplicada.
¿Por qué no veo el "ícono de martillo"?▼
En versiones recientes de Claude Desktop el acceso a los conectores MCP no es un martillo, sino el botón "+" en el chat. Haz clic en "+" y selecciona "Conectores" — ahí encontrarás powerbi-modeling-mcp con un toggle para activarlo.
¿Puedo usarlo desde el navegador (claude.ai)?▼
No para Power BI Desktop local. El MCP corre en tu máquina y el navegador no puede acceder a servicios locales. Si publicas tus reportes en Power BI Service (nube de Microsoft), existe una integración remota que sí funciona desde el navegador, pero requiere licencia Pro.
¿Qué pasa si Claude modifica algo que no quería?▼
Por eso recomendamos empezar en modo "--readonly". Además, Power BI guarda historial de cambios — usa Ctrl+Z para deshacer o cierra el archivo sin guardar para volver al estado anterior.
¿Funciona con cualquier versión de Power BI Desktop?▼
Funciona con versiones recientes de Power BI Desktop (2024 en adelante). Asegúrate de tener la versión actualizada desde el sitio oficial de Microsoft o desde la Microsoft Store.
